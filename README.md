# Graveyard of Marksmen

5×5盤で向きと射線を読み合う2人用アブストラクトゲーム  
**Graveyard of Marksmen** の無料ブラウザ版です。

## 遊び方

`index.html` をブラウザで開くと遊べます。

## 公開方法

### GitHubにアップロードする場合

このリポジトリの直下に `index.html` を置いてください。

### Cloudflare Pagesで公開する場合

Cloudflare Pages の設定例です。

- Framework preset: `None`
- Build command: 空欄、または `exit 0`
- Build output directory: `/`
