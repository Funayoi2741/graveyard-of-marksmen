# Graveyard of Marksmen

5×5盤で向きと射線を読み合う2人用アブストラクトゲームです。  
倒された駒は盤上から消えず、その場に「墓標」として残ります。

## 概要

Graveyard of Marksmen は、5×5の盤面で遊ぶ2人用の対戦ゲームです。  
駒は現在の向きに従って攻撃し、倒された駒は盤上から取り除かれず、墓標として移動や攻撃を遮ります。  
相手の司令を墓標にすることが主な勝利条件です。
